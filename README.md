# 🌙 Houda's Cosmic Capsule 

A Time Capsule Web Application

## 🌐[live Website](https://hodacosmiccapsule.netlify.app/)

## 📚 About This Project 

This is a web application where users can create time capsules that are locked until a future date. Users can save their memories , photos , and messages to open later  .

## 📌Why I Created  This💡

Last year, our school had an activity where we wrote letters to ourselves .We were told to come back in 6 years and open them ! .I wrote my letter, sealed it, and waited. But I realized... what if i cant go back to school in 6 years? What if the paper gets lost ?That's when I decided to build this app .**A Digital Place That is Always With You!**

## ✨ Features 
- Create Time Capsules 
- Lock/Unlock System 
- Photo Upload 
- Save to Favorites
- Statistics Page
- Theme Toggle (Dark/Light)
- Responsive Design
- Notifications (5 minutes before unlock)
- Auto save (where the user writes the letter)
- Interactive Audio: Featuring a **magical** sound effect upon successfully locking a capsule 

## 💻 Technologies Used 
- **HTML**
- **CSS**
- **JavaScript**
- **LocalStorage**

## 🚀🌌 How To use 🚀
1. Open the website
2. Create a time capsule 
3. Choose an unlock date 
4. Add your message and photo(optional)
5. Wait for the unlock date 
6. Open your capsule !




### ⚡Future Improvements 
- **AI Assistant:** Integrate an AI API to provide writing prompts, offer support to users as they write, and help them reflect on their current journey .
- **Emotional Analytics:**Use AI to analyze the "mood" of each capsule and save these emotional snapshots to track personal growth over time .
- **Multi-language Support:**Integrate a translation toggle (e.g, Arabic)to make the "Cosmic Capsule" accessible to a global audience .
- **Theme Personalization:**Implement a style switcher with 5 or more distinct color palettes so users can customize their experience.
- **Cloud Synchronization:**Transition from Local Storage to cloud database so users can access their capsules from any device or browser!
- **Advanced Media Attachments:**Expand the upload feature to support voice notes or short video messages for a more immersive memory .
