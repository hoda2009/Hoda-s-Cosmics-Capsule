# 🌙 Houda's Cosmic Capsule 

A Time Capsule Web Application

## 🌐[live Website]()

## 📚 About This Project 

This is a web application where users can create time capsules that are locked until a future date. Users can save their memories , photos , and messages to open later  .

## 📌Why I Created  This 

Last year, our school had an activity where we wrote letters to ourselves .We were told to come back in 6 years and open them ! .I wrote my letter, sealed it, and waited . But I realized... what if I could open it sooner ? What if I could create my own time capsule anytime ?That's when I decided to build this app .

## ✨ Features 
- Create Time Capsules 
- Lock/Unlock System 
- Photo Upload 
- Save to Favorites
- Statistics Page
- Theme Toggle (Dark/Light)
- Responsive Design
- Notifications (5 minutes before unlock)

## 💻 Technologies Used 
- **HTML**
- **CSS**
- **JavaScript**
- **LocalStorage**

## 🚀 How To use 🚀
1. Open the website
2. Create a time capsule 
3. Choose an unlock date 
4. Add your message and photo(optional)
5. Wait for the unlock date 
6. Open your capsule !
